== Description ==
Enables you to optionally push users to your [AWeber](http://aweber.com "AWeber") email list when they signup through mingle... This plugin requires you to also be running [Mingle](http://wordpress.org/extend/plugins/mingle "Mingle").

= See Mingle In Action! =
If you're wondering what Mingle is -- it is another WordPress plugin that easily turns your wordpress blog into a social network. You can now try Mingle out in our Mingle Community! Come be my friend at:

[Mingle WordPress Community](http://wpmingle.com "Mingle WordPress Community")

== Installation ==
1. Make sure Mingle is already installed

2. Upload 'mingle-aweber-signup.zip' to the '/wp-content/plugins/' directory

3. Activate the plugin through the 'Plugins' menu in WordPress

4. Go to "Mingle" -> "Options" in your WordPress admin and enter the name of the [AWeber](http://aweber.com "AWeber") list that you want users to sign up for
